# Plots a distributed lag function for model summary of 'tdlm'

Method for plotting a distributed lag function for model summary of
'tdlm'

## Usage

``` r
# S3 method for class 'summary.tdlm'
plot(x, ...)
```

## Arguments

- x:

  object of class 'summary.tdlm', output of summary of 'tdlm'

- ...:

  additional plotting parameters for title and labels 'start.time' which
  sets the first time value

## Value

A plot of distributed lag effect estimated with tdlm

plot.summary.tdlm

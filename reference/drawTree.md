# Draws a new tree structure

A recursive method for drawing a new tree structure

## Usage

``` r
drawTree(depth, alpha, beta)
```

## Arguments

- depth:

  depth of a tree

- alpha:

  tree shape parameter, 0 \< alpha \< 1

- beta:

  tree size parameter, beta \> 0

## Value

integer value of number of terminal nodes

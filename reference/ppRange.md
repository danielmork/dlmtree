# Makes a 'pretty' output of a group of numbers

Method for making a 'pretty' output of a group of numbers. For example:
2,3,4,5,8,9,12,15,16 becomes 2-5,8-9,12,15-16

## Usage

``` r
ppRange(r)
```

## Arguments

- r:

  set of integers to make 'pretty'

## Value

character string of values representing 'r'

## Details

ppRange

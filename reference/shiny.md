# shiny

shiny generic function for S3method

## Usage

``` r
shiny(fit)

# S3 method for class 'hdlm'
shiny(fit)

# S3 method for class 'hdlmm'
shiny(fit)
```

## Arguments

- fit:

  object of class 'hdlm', 'hdlmm' to which S3method is applied

## Value

shiny interface for further analysis on heterogeneous analyses. The
interface includes tabs for modifier selection, personalized exposure
effects and subgroup-specific effects.
